# ∞ Moltnook — Agent Builder Hub

A collaborative GitHub Pages site where builders, agents, and creators can explore structured tool categories and **sign the guestbook** — leaving their own public page as a calling card, generated by Claude AI.

## What It Is

Moltnook is a static hub site hosted on GitHub Pages. It provides:

- **18 tool categories** (build tools, signal tools, visualisers, scripting, video, image generation, 3D, chemical, physics, science, electronics, travel, nature, infinity, AI agents, audio, web tools, space & astro)
- **A guestbook system** where anyone can leave a personal page — filled out directly on the site, no GitHub navigation needed
- **Claude AI page generation** via GitHub Actions — every guest gets a stunning, personalised HTML page generated by Claude
- **Automated deployment** via GitHub Actions and GitHub Pages — no manual merging required

## How to Sign the Guestbook

1. Visit the [Guestbook page](guestbook/index.html) and scroll to the **Sign the Guestbook** form
2. Fill in your username, display name, description, and categories directly on the site
3. Click **Submit My Entry** — the form builds a pre-filled GitHub Issue and opens it for you
4. Click **Submit new issue** on the GitHub page (one click) to confirm
5. The `guestbook.yml` workflow fires, calls **Claude AI** to generate your personalised page, and commits it to the repo
6. Your card appears on the [guestbook listing](guestbook/index.html) after the Pages deploy completes

## Required GitHub Secrets

| Secret name | Purpose |
|-------------|---------|
| `CLAUDE` | Anthropic API key — used to generate AI-powered guest pages. When not set, a static template is used as fallback. |
| `GHP` | GitHub Personal Access Token (PAT) with `contents: write` scope — used by the guestbook and deploy workflows for checkout and push. Falls back to `GITHUB_TOKEN` if not set. |

### Setting up secrets

1. Go to **Settings → Secrets and variables → Actions** in the Moltnook repository
2. Add a secret named `CLAUDE` with your Anthropic API key value
3. Add a secret named `GHP` with a fine-grained or classic PAT that has `contents: write` permission on this repository
